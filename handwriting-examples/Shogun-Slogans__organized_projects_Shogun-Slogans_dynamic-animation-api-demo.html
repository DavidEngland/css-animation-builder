üé® Usage Examples

WordPress Shortcode

[shogun_animation type="typewriter" text="Hello World!" speed="80" color="#ff6b6b"]

What happens:

  1. PHP generates unique CSS for this specific configuration
  2. CSS is cached for reuse
  3. HTML is rendered with unique class
  4. Minimal JS initializes the animation

REST API Call

POST /wp-json/shogun-slogans/v1/generate-css { "animation": "typewriter", "parameters": { "speed": 80, "color": "#ff6b6b", "cursor": "‚ñå" } }

Response:

{ "css": ".shogun-typewriter-abc123 { --typing-speed: 80ms; --text-color: #ff6b6b; }", "cache_key": "abc123", "js_init": "ShogunAPI.initTypewriter", "selector": ".shogun-typewriter-abc123" }
