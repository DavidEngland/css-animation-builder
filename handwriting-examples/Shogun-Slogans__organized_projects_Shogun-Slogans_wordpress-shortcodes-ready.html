üìù Using in Block Editor:

  • Add a Custom HTML block
  • Paste any shortcode from above
  • Preview or publish to see the animation

üìù Using in Classic Editor:

  • Switch to Text mode
  • Paste the shortcode directly
  • Switch back to Visual mode if needed

‚öôÔ∏è Customization Options:

  • text: The text to display
  • speed: Typing speed in milliseconds (lower = faster)
  • cursor: The cursor character (|, _, ‚ñå, ‚òÖ, üî•, etc.)
  • loop: "true" or "false" for repeating animation
  • delay: Delay before starting in milliseconds
  • style: typewriter, handwritten, gradient, neon, rainbow, terminal
  • animation: fade, slide, bounce, typewriter
