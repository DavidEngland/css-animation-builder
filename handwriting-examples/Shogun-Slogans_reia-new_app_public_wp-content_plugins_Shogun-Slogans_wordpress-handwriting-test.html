📋 WordPress Shortcodes for Handwriting Effects

🖋️ Quill Writing Style

[typewriter_text text="We will help you make The Smart Move — I guarantee it!" speed="120" cursor="🖋️" class="quill-writing" style="font-family: 'Caveat', cursive; font-size: 2rem; color: #2c3e50; text-align: center;"]

✒️ Fountain Pen Style

[typewriter_text text="We will help you make The Smart Move — I guarantee it!" speed="140" cursor="🖋️" class="fountain-pen-writing" style="font-family: 'Dancing Script', cursive; font-size: 2.5rem; color: #1a1a2e; text-align: center;"]

🎨 Using Custom HTML Block in WordPress

Copy this HTML into a Custom HTML block:

<div class="shogun-typewriter quill-writing" data-text="We will help you make The Smart Move — I guarantee it!" data-speed="120" data-cursor="🖋️" data-loop="true" style="font-family: 'Caveat', cursive; font-size: clamp(1.5rem, 3.5vw, 2.5rem); color: #2c3e50; text-align: center; padding: 2rem;"> <span class="typewriter-text"></span> <span class="typewriter-cursor">🖋️</span> </div>

📝 Step-by-Step WordPress Instructions

  1. Make sure the Shogun Slogans plugin is activated
  2. Edit a page or post in WordPress
  3. Add a Shortcode block and paste one of the shortcodes above
  4. OR add a Custom HTML block and paste the HTML code
  5. Preview or publish to see the handwriting typewriter effect!

✨ Available Classes

  • quill-writing - Floating quill pen with ink flow effect
  • fountain-pen-writing - Elegant fountain pen with Spencerian ink effects
  • shogun-handwriting-caveat - Forces Caveat font
  • shogun-handwriting-dancing - Forces Dancing Script font
  • shogun-handwriting-tangerine - Forces Tangerine font

🎯 Tips for Best Results

  • Use speed between 120-140ms for elegant handwriting feel
  • The 🖋️ cursor works best for both quill and fountain pen styles
  • Use center alignment for dramatic effect
  • Add padding for better visual spacing
  • Use clamp() for responsive font sizes
