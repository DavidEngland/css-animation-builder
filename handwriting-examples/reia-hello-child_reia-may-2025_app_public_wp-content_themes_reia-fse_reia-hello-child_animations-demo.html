üìù Implementation Notes

  • Scroll Animations: Some animations (like basic-anim-scroll-reveal) require JavaScript to detect when elements enter the viewport.
  • Accessibility: All animations respect the prefers-reduced-motion setting for users who prefer reduced motion.
  • Performance: Animations use hardware acceleration (transform and opacity) for smooth performance.
  • Customization: Use modifier classes like basic-anim-delay-medium and basic-anim-duration-fast to customize timing.
