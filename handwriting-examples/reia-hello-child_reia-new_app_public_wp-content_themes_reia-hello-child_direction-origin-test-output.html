Available Direction Values

  • normal - Default direction (forward)
  • reverse - Animation plays backwards
  • alternate - Animation alternates forward/backward each iteration
  • alternate-reverse - Animation alternates backward/forward each iteration

Available Transform-Origin Values

  • center - Default center point
  • top, bottom, left, right - Edge-based origins
  • top-left, top-right, bottom-left, bottom-right - Corner origins
  • Custom values - e.g., '25% 75%', '10px 20px'

Performance Notes

  • Animation direction is controlled via CSS custom properties for easy runtime modification
  • Transform-origin is set as a static property on animation classes
  • Utility classes provide alternative control methods
  • Direction is particularly useful for infinite and repeating animations
  • Transform-origin greatly enhances visual effects for rotation and scaling
